# SaludConecta

Una aplicación moderna y elegante para la gestión de citas médicas.

## Características

- Dashboard para Pacientes, Doctores, Recepcionistas y Administradores.
- Búsqueda y reserva de citas.
- Gestión de perfiles y horarios.
- Interfaz moderna con Tailwind CSS y Framer Motion.

## Cómo ejecutar

1. Instala las dependencias:
   ```bash
   npm install
   ```

2. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Para verlo en tu red local (celular):
   ```bash
   npm run dev -- --host
   ```
